CSS Flexbox demo

Webpages are set up with rows and columns so that the design lines are straight and clean. This is pleasing to the human eye. This is why you should never, EVER, (hardly ever) center-align a block of text. But that is a lesson for a different day.

I have organized the layout of these samples in rows, but you can equally organize in columns. See this example from W3Schools.

Do not forget to control the page layout with a main container! see line 42 of this code. Also - all this style information should be on your attached CSS, not in the header of your document. I just put it there so that I coud have this all on one page.

Feel free to use any of this code.
